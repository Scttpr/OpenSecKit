# NIS2

Directive (UE) 2022/2555 - Cybersécurité.

!!! warning "En cours de développement"
    Le support NIS2 est en cours d'implémentation.

## Vue d'ensemble

La directive NIS2 renforce les exigences de cybersécurité pour les entités essentielles et importantes dans l'UE.

## Entités Concernées

### Entités Essentielles

- Énergie
- Transport
- Banque
- Infrastructure des marchés financiers
- Santé
- Eau potable
- Eaux usées
- Infrastructure numérique
- Gestion des services TIC (B2B)
- Administration publique
- Espace

### Entités Importantes

- Services postaux
- Gestion des déchets
- Chimie
- Alimentation
- Fabrication
- Fournisseurs numériques
- Recherche

## Exigences (Art. 21)

| Domaine | Description |
|---------|-------------|
| Gestion des risques | Analyse et traitement des risques |
| Gestion des incidents | Détection, réponse, notification |
| Continuité | PCA/PRA |
| Supply chain | Sécurité des fournisseurs |
| Développement | Sécurité des systèmes |
| Évaluation | Tests et audits |
| Cryptographie | Chiffrement approprié |
| RH | Formation et sensibilisation |
| Contrôle d'accès | MFA pour accès critiques |

## Notification d'Incident (Art. 23)

| Délai | Action |
|-------|--------|
| 24h | Alerte précoce |
| 72h | Notification initiale |
| 1 mois | Rapport final |

## Sanctions

- Entités essentielles : jusqu'à 10M€ ou 2% CA mondial
- Entités importantes : jusqu'à 7M€ ou 1.4% CA mondial
